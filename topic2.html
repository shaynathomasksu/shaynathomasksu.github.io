Origins of HTTP

HTTP was introduced in 1989 by Tim Berners-Lee as part of the original design for the World Wide Web. The initial version, HTTP/0.9, was a minimal protocol allowing clients to request a single HTML document using only the GET method.

HTTP Versions and Improvements

Version Year Released Key Features
HTTP/0.9 1989 Simple GET requests, no headers or status codes
HTTP/1.0 1996 Introduced headers, status codes, POST and HEAD methods
HTTP/1.1 1997 Persistent connections, chunked transfer, caching
HTTP/2 2015 Multiplexing, header compression, binary framing
HTTP/3 2022 Uses QUIC (UDP), faster and more reliable connections
